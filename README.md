# MyCredential

Mobile application that allows users to register for events, access their credentials, and use a QR code for check-in on the event day.

## Techs

- React Native (Expo)
- TypeScript
- NativeWind
- Axios
- Node.js (Fastify)
- Prisma
- SQLite

## Setup

Clone this repository:

